Instruction-Set Simulator (ISS)

The ISS gives you access to additional capabilities that are not available on hardware, including the ability to:

  • set hardware watchpoints on memory and registers
  • specify memory size and location
  • stop execution on loads of a value
  • trace instructions
  • view the most recently executed instructions

See the Debugger User's Guide for more information about the ISS.
