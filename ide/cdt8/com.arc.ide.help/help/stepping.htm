Stepping through a History of Instructions

Use the stepping buttons to step forwards or backwards through source-level statements or assembly:

The step buttons work the same way as the stepping toolbar buttons.

Double click an instruction to view an Instruction history disassembly window and a Register set from trace window that shows the value of the registers.

Register entries marked with "--------" are registers for which no history exists.

The Register set from trace window appears only if the history of instructions includes register write events.
