Entering Input to the Application Being Debugged

You can enter input into the application you are debugging using the Console view as stdin.

Example
  1. Open your application source in the Editor view.
  2. Enter the following code: 
#include stdio.h
char c;
c = getc(stdin);
putc (c, stdout);
fflush(stdout);
  1. Save the file and select Project > Build All.
  2. Select or specify a debug configuration, and start debugging.
  3. Step until you reach c = getc(stdin);
  4. Click in the Console view and enter a character, then press Enter.
  5. Step again.
  6. Open a Variables window and note the value of c.
  7. Step until the character you entered is echoed to the screen.
