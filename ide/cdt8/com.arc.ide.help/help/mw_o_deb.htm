Debug Views and Windows

The Debug perspective offers two ways to access many of the debugging functions available in the IDE:

  • Eclipse views from Window > Show View
  • Debugger-supplied windows from the Debugger menu.

Both methods rely on the underlying debugger engine. They let you manipulate the same information in slightly different ways.
