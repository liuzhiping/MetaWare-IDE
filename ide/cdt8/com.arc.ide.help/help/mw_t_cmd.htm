Using Debugger Commands

You can use a Commands view to issue command-line commands to the debugger.

  1. Open a Commands view, select Debugger > Command-line Input when your application is running in the debugger (the Debugger menu is present in the menubar).
  2. Enter any valid debugger command at the prompt:

The command and any printed output appear in the Console view, preceded by the debugger command prompt. The drop-down list shows a history of commands that is preserved between sessions.

To clear the history list, click the Clear command history button:

To print a list of debugger commands, type help summary or see Debugger Commands.
