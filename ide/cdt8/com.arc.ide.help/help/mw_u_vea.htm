VideoCore 16-Bit Vector-Registers Window

For VideoCore targets, you can open a Vector registers (16-bit) window in the Debug perspective by selecting Debugger > 16-bit vec registers when your application is running in the debugger.

The Vector registers (16-bit) window shows the VideoCore vector-register file as if it were an array of 16-bit values. The columns are titled with the word index into each vector. The rows are titled with the vector number.

In this 16-bit view, 8-bit values are paired to produce 16-bit values. The pairing is not trivial, but if you program the vector-register file directly, the pairing is more easily understood. The VideoCore Processor Architecture Manual explains the pairing of 8-bit values.

CAUTION
Watchpoints on vector registers do not correctly display changed values.
