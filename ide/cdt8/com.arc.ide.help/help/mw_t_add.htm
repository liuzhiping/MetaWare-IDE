Adding Files to a Project

You can add one or more source files to a project in the C/C++ perspective in several ways:

  • Create a new file using File > New > File while in a project (or right-click a project and select New > File).
  • Drag one or more files from a folder and drop them into your project.
  • Import:
  • Select File > Import while in a project (or right-click the project and select Import).
  • Expand the General node.
  • Select File system and click Next.
  • Type or browse to the folder where the files are located and click OK. A dialog appears in which you can select files to import. You can make additional choices in the Options area.
The dialog might contain an error message until you select files to import.
  1. When you have made your selections, click Finish.

The IDE imports the file and returns to the C/C++ perspective.
