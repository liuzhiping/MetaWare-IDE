Debugging with Breakpoints

  1. Ensure prime.elf is loaded in the Debug perspective (see Building and Debugging). If you are already debugging, click the Restart button in the Debug view:
  2. Set a breakpoint by double-clicking the frame of the Editor view left of the line printf("%d Prime Numbers between 3 and 16383:\n",Count);

    A marker in the Editor view frame next to the line of code shows that a breakpoint is set.
You can also set breakpoints by double-clicking in the Debugger > Source window, or the Debugger > Disassembly window.
  1. Run to your breakpoint.
