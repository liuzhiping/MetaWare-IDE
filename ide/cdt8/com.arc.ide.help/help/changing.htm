Changing Project Type (Executable or Shared Library)

You can change an existing project from an executable to a shared library or from a shared library to an executable by editing the tools (linker or archiver) and changing the artifact type.

  1. Select Project > Properties, expand C/C++ Build, and select Tool Chain Editor.
  1. Click the Select Tools... button.
    The Select tools dialog appears.
  1. Check Allow all changes to enable the buttons for changing tools.
  2. For a shared library, remove the linker and add the archiver for your target processor; for an executable, remove the archiver and add the linker for your target processor.
  3. Click OK to return to the Properties dialog.
  4. In the Properties dialog, Select C/C++ Build > Settings from the tree menu.
  5. Select the Build Artifact tab.
  1. Change the Artifact Type combobox to Shared Library or Executable
  2. Click OK.
