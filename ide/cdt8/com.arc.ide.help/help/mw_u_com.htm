Debugger: COMM-Port Simulator Settings

This section lists the settings you may make in the COMM Port Simulator area of the Debug Configurations dialog. For information about accessing the dialog, see Specifying a Terminal or COMM-Port Simulator.

If you select COMM Port Simulator, use the check boxes and drop-down lists to specify the following for your COMM-port simulator:

Make settings here only if you wish to change the defaults.
  • Specify the Receive buffer size (in bytes).
  • Specify the Transmit buffer size (in bytes).
  • Specify Baud rate. Select a value at or below the maximum supported by your hardware.
  • Specify the Parity.
  • Specify the Stop bits.
  • Specify the Flow control.
  • Specify the Word length (data bits: ASCII is 7; binary is 8).

To specify more than one COMM-port simulator, specify the properties of each additional simulator on the command line using -simextp. You can use the -simextp=install_dir/bin/commsim entry in the Debugger Options: field at the bottom of the dialog as an example.
