Animation

Animation is a way to automate stepping, one statement or instruction at a time, pausing between individual source statements or machine instructions (animation speed is configurable).

Animation stops automatically at these points:

  • when execution encounters a breakpoint
  • when a watchpoint's value changes
  • when you click the Terminate button
  • when the process terminates
Animation is the only method of execution that stops automatically when the value of a software watchpoint changes.

The Animation toolbar appears only when selected; for information on using animation, see Debugging: Animating a Process.
