Overlays

For ARCompact and ARCv2 targets, MetaWare C provides an Automated Overlay Manager to support locating functions and data at the same address in RAM at different times using overlays. See the Automated Overlay Manager User's Guide for details on coding and building applications that use overlays.

3.2.1.1 Developing Applications That Use Overlays

To develop applications that use overlays, do the following:

  1. See the Automated Overlay Manager User's Guide for keywords and other important coding and build considerations.
  2. Specify overlay-related build options in the Properties > Settings > Tool settings > Overlay Support for your project.
3.2.1.2 Debugging Applications That Use Overlays

The MetaWare debugger automatically understands applications that use the Automated Overlay Manager and takes care of the switching without distractive details, while providing the functionality to display which overlays are currently loaded.

For information about debugging applications that use overlays, see Debugging with Overlays (ARCompact and ARCv2).
