Using a Digilent Cable to Connect to Hardware

You can connect the debugger to an ML50x development board using a Digilent cable.

Do the following to debug on an ML50x board using a Digilent cable:

  1. In the C/C++ Projects view, select a project.
  2. Open the drop-down menu next to the Debug button and select Debug Configurations....
  3. Click the Debugger tab.
  4. Ensure that MetaWare Debugger is selected in the drop-down menu.
  5. In the Debugger Options tree view, select Target Selection.
  6. Select Hardware and Digelent JTAG cable from the drop-down lists:
A standard four-wire connection is used by default. If your hardware has a two-wire JTAG interface, you can use the JTAG wire protocol drop-down to select two ways to use it:
  • 2-wire connects only if the two-wire JTAG connection is available.
  • 2-if-4-fails tries a four-wire connection first, then tries a two-wire connection if four-wire fails.
  1. Click OK after making your selections.
  2. Click Debug to start debugging.
Switching Dynamically While Debugging

You can use the prop command to switch between two-wire and four-wire mode dynamically while debugging:

  • Use prop jtag_wire=2 or jtag_wire=4 to shift between two-wire and four-wire connections.
  • Use prop jtag_status to display the current JTAG status (two-wire or four-wire connection).

You can use the prop command on the command line or in the Command: input field.
