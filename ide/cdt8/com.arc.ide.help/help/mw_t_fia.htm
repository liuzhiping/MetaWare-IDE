Copying a File to Memory

Open the File/Memory/Fill operation dialog by selecting Debugger > File-memory-fill operation... when your application is running in the debugger (the Debugger menu is present in the menubar).

  1. Select Copy a file into memory.
  2. If you are loading an S-record file, select The file is an S-record file.
  3. Specify the beginning memory address in the Address field.
  4. Enter or browse to the file containing the values to read using the File to read field.
  5. Click OK to copy.
