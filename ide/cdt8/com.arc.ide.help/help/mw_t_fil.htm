Filling Memory with a Value or String

You can fill a block of memory with arbitrary values using the File/Memory/Fill operation dialog. You can use this option to initialize larger blocks of memory with the same byte value, hexadecimal string, or character string.

To access the File/Memory/Fill operation dialog, select Debugger > File-memory-fill operation... when your application is running in the debugger (the Debugger menu is present in the menubar).

  1. Select Fill memory.
  2. Enter the starting address in the Starting address field.
  3. Enter the total size in bytes to fill in the Length field.
  4. For a single value, select the value size to repeat and enter the value in the value field.
  5. For a string, select the type of string and enter the string in the string field.
  6. Click OK to fill memory.
