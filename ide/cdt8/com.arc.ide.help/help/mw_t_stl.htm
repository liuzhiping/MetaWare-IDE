Selecting C++ Run-Time Libraries

A minimal MetaWare C++ Run-Time Library is linked in by default. You can choose to compile and/or link with the STLport Standard Template Library for a full C++ STL.

Compiling and Linking

To compile and link with the STLport STL, select Use STLPort C++ Library (-Hstl) in the C++ category in Project > Properties > Settings, Tool Settings tab, MetaWare ARC C/C++ Compiler. By default, a smaller, faster version without exceptions is supplied; for exception-aware libraries, additionally select Generate Exception-Aware Code (-Hexcept).

Linking

To link in the STLport STL without compiling, select your preferred version (with or without exceptions) from the Runtime C++ library drop-down list in Project > Properties > Settings, Tool Settings tab, MetaWare Linker for ARCompact > Special Libraries.

You can make other library-related linking selections here as well.
