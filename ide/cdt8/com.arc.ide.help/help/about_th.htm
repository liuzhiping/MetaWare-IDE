About This Document

The MetaWare IDE is an extension to the Eclipse IDE. By the same token, this document is an extension to the Eclipse C/C++ Development User Guide in the IDE online help. Look to the Eclipse C/C++ Development User Guide for general ways of doing things, and look here for MetaWare variations.

This document presents the MetaWare extensions divided into the same sections as the Eclipse C/C++ Development User Guide in the IDE online help:

  • Getting Started
  • Concepts
  • Tasks
  • Reference

The extensions are then further divided, where applicable:

  • Workbench extensions
  • Compilation extensions
  • Debugging extensions
