Using Synopsys Virtualizer

You can select the Synopsys Virtualizer debug server as the debug target. Choose this target when you are using a Synopsys Virtualizer SystemC model of your ARC system.

If you are using the nSIM Synopsys Virtualizer SystemC implementation, you do not have to set up a SCIT connection if you select Synopsys Virtualizer as the target.

You can select Synopsys Virtualizer using the Debugger Options tree view:

  1. In the C/C++ Projects view, select a project.
  2. Open the drop-down menu next to the Debug button and select Debug Configurations....
  3. Click the Debugger tab.
  4. Ensure that MetaWare Debugger is selected in the drop-down menu.
  5. In the Debugger Options tree view, select Target Selection.
  6. Select Synopsys Virtualizer from the Select the target drop-down list:
  7. Click OK after making your selections.
  8. Click Debug to start debugging.
