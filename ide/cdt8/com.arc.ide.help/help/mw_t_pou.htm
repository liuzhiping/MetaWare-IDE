Working with Application Output

By default, screen output from your application is displayed in the Console view. To display program output in a dedicated window with additional functionality, open a Program output window by selecting Debugger > Program output when your program is running in the debugger.

The Program output window is not available for all targets.
  • To filter output, type a regular expression in the Reg expn text box and press the Enter key.
  • To show or hide line numbers, click the button.
  • To clear the screen, click the Discard button.
  • To save your output to a file, right-click and select Save to File.
