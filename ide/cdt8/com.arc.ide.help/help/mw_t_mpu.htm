Viewing the MPU Registers

You can enable a window for the Memory Protection Unit (MPU) registers for hardware or simulator for target processors that support an MPU.

4.11.5.1 Enabling the MPU registers Window
  1. Click the downward arrow next to the Debug button and select Debug Configurations... from the drop-down menu:
  2. In the resulting Debug Configurations dialog, select the Debugger tab.
  3. Select Simulator Extensions > MMU/MPU Support in the Debugger Options tree view.
  4. Check Enable MPU.
  5. Make any other selections you require and click Debug to start debugging.
  6. If a Confirm Perspective Switch window appears, click Yes.

A Debug perspective opens for your application.

4.11.5.2 Opening the MPU registers Window

To open the MPU registers window after it is enabled, select Debugger > Hardware > MPU registers after your application is running in the debugger.

If you are already debugging the application, you must click the Restart button for your changes to take effect.

You can expand nodes in the tree view down to the bit level. For information on the register names and contents, see the ARC 600 Programmer's Reference Manual.
