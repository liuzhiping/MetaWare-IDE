Using the Cycle-Accurate Simulator (ARC 600 Only)

The Cycle-Accurate Simulator (CAS) is a simulation tool for assisting in the accurate analysis and optimization of ARC 600 processor configuration options and extensions. It connects to the debugger like other simulator DLLs.

You specify target DLLs on the Debugger tab when setting up a debug configuration.

For more information, see the CAS User's Guide for your ARC processor.
