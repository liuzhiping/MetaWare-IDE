Defining MetaWare Build Settings

By default, the IDE is preconfigured with the most useful build settings (options and toggles) for your target processor. You can change these options and toggles and specify new options, toggles, and pragmas. The following topics tell you how.

  • Specifying Options, Toggles, and Pragmas Using the GUI
  • Manually Adding Options, Toggles, and Pragmas
  • Adding Assembler and Linker Options to the mcc Driver
  • Applying Settings to a Single Source File
  • Specifying Options, Toggles, and Pragmas for Makefile Projects
  • Selecting C++ Run-Time Libraries
4.2.2.1 Specifying Options, Toggles, and Pragmas Using the GUI

To specify options, toggles, and pragmas to the compiler, linker, and assembler for your project, do the following:

  1. Select Project > Properties for your project.
  2. Expand C/C++ Build and click Settings.
  3. Change the pre-specified options and add new ones using the dialogs that appear when you click the items in the Tool Settings tab.
4.2.2.2 Manually Adding Options, Toggles, and Pragmas

You can also make settings manually. This is handy if you know the name of the option ("flag"), toggle, or pragma from the listings in the toolkit manuals:

  • MetaWare C/C++ Programmer's Guide
  • ELF Assembler User's Guide
  • ELF Linker and Utilities User's Guide
Specifying the Controls
  1. On the Tool settings tab, click General.
  2. Click the Add button in Additional Compiler Flags or Pragma specifications.

Here you can specify any of the mcc compiler and driver controls acceptable on the command line.

Additional Compiler Flags
    • For options, click the Add... button in the Additional Compiler Flags area and type the option in the Enter Value dialog that appears.
    • If the option you specify requires macro-like variables such as the path of the project or workspace, you can use the Build Variable... button to reference them.
    • For toggles, use option -Hon=toggle_name
      or -Hoff=toggle_name.
    • If the toggle you specify requires macro-like variables such as the path of the project or workspace, you can use the Build Variable... button to reference them.
Pragma Specifications
    • For pragmas, click the Add... button in the Pragma specifications [e.g., offwarn(123)] area and enter pragma and its parameters in the Enter Value dialog that appears.
    • If the pragma you specify requires macro-like variables such as the path of the project or workspace, you can use the Build Variable... button to reference them.
4.2.2.3 Adding Assembler and Linker Options to the mcc Driver

Technically, mcc is a driver program that controls the compiler, linker, and assembler.

  • You can manually add assembler options under ARCompact Assembler > General.
  • You can manually add linker options under MetaWare Linker > Miscellaneous.
4.2.2.4 Applying Settings to a Single Source File

Some options, toggles, and pragmas can be applied to some source files while excluding other files. To apply a setting to one file only, proceed as follows:

  1. Right-click the source file in the C/C++ Projects pane and select Properties.
  1. Select C/C++ Build > Settings.
  2. Select an item in the Tool Settings tab and make your file-specific settings in the dialog that appears for that item.
  3. Click OK.
  4. Repeat for other files as necessary.
4.2.2.5 Specifying Options, Toggles, and Pragmas for Makefile Projects

For makefile projects, add the options, toggles and pragmas to the build command in your makefile (for toggles, use -Hon=toggle_name; for pragmas, use -Hpragma=pragma_name).

Obtaining Listings from the Toolkit Manuals

For lists of available options, toggles, and pragmas, see the following manuals:

  • MetaWare C/C++ Programmer's Guide
  • ELF Assembler User's Guide
  • ELF Linker and Utilities User's Guide

Obtaining Listings from the Tools

You can also view a listing of available options, toggles, and pragmas using the command-line help for each tool.

  • For a list of compiler options, toggles, and pragmas, type -h on the driver command line, for example: 
       mcc -h
  • For a list of assembler options, type -h on the astarget command line, for example: 
       asac -h
  • For a list of linker options, type -h on the ldtarget command line, for example: 
ldac -h
