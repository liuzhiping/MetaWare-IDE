Viewing MMU Contents

The MMU contents window displays the contents of the MMU in sets.

You must specify ARC 700 and MMU in the Debug Configurations dialog for this window to be available.

  • Click the button to update the display.
  • Content: is a button you can use to cycle between three types of content:
    • all - all MMU contents
    • NZ - only sets with one or more non-zero entries
    • valid - only sets with one or more valid entries
  • Enter an ASID in the ASID field and push Enter to show only sets with that ASID.
