Global and Local Variables

The MetaWare IDE divides variables into global and local variables. The Eclipse Variables view includes global variables if they are directly in scope at the PC location, while the debugger considers any variable within scope to be local, even if the variable is static or global. Thus you might not be able to delete global variables from the Eclipse Variables view if the MetaWare debugger has classified them as part of the stack-frame context.
