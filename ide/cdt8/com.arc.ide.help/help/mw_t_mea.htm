Copying Memory to a File

You can copy the contents of memory to a binary file using the File/Memory/Fill operation dialog. You can use the file later to initialize memory with values using the copy-file-to-memory function.

To access the File/Memory/Fill operation dialog, select Debugger > File-memory-fill operation... when your application is running in the debugger (the Debugger menu is present in the menubar).

  1. Select Copy memory to file.
  2. Enter the starting address in the Address field.
  3. Enter the total size in bytes of the memory you want to copy in the Length field.
  4. Enter or browse to the file in the File to write field.
  5. Click OK to save memory to the file.
