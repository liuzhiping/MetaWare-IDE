Image Memory Window

The Image memory window shows the data in an image as if that data were memory, even if the image source is not memory. For example, the image source might be a vector-register file.

The columns are titled with the offset from the address given on the left edge.

The window shows its contents in the default size of the image element:

  • 1 byte
  • 2 bytes
  • 4 bytes
