Working with a Hardware Stack

The Hardware Stack window shows memory with a display width the same as the natural size of elements on the run-time stack. It also shows the stack-pointer register at the line whose address equals the value of the stack pointer.

To open a Hardware Stack window, select Debugger > Hardware stack when your program is running in the debugger.

4.15.2.1 Changing the Display

You can do any of the following using the toolbar:

  • Use the Goto: field to enter the address of a memory block to display.
  • Change the base numbering system: in the Base field, choose 16, 10, 8, or 2.
  • Change how the contents are grouped: in the Size field, choose the data type by which you want the bytes of memory to be grouped.
  • Change the order of the display: use the arrow buttons to toggle between ascending and descending address order.
  • Change the value of a memory address using the Change button (or right-click a line and select Change).

To open an Examine window for a variable on the stack, right-click and select Examine.
