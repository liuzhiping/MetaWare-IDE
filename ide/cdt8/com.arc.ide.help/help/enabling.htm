Enabling the PMU Window

On hardware, the presence of a PMU is automatically detected; on simulators, you must enable it.

  1. Select your ELF executable in the Project Explorer in the C/C++ perspective.
  1. Click the downward arrow next to the Debug button:
  2. Select Debug Configurations... from the drop-down menu:
The Debug Configurations dialog appears.
  1. Select C/C++ Application.
  2. Click the New launch configuration button:
The Debug Configurations dialog opens with a default name for your configuration in the Name: text box
  1. Change the name of your debug configuration if desired.
  2. Select the Debugger tab if it is not already selected.
  3. If you are using xISS, select Target Selection in the Debugger Options tree view and select Fast ISS (xISS) from the Select the target drop-down list:
  4. Select Simulator Extensions > Power Management Unit in the Debugger Options tree view.
  5. The right side of the dialog changes to show PMU settings. Check the Enable Power Management Unit (PMU) checkbox:
The option -pmu now appears in the list of Debugger Options.
  1. Click Apply.
  2. Make any other selections you require and click Debug to launch your debug configuration.
  3. If a Confirm Perspective Switch window appears, click Yes.

A Debug perspective opens for your application. To open the PMU window, see Opening the PMU Window.

If you are already debugging the application, you must click the Restart button for your changes to take effect.
