Debugger Commands

The following is a list of debugger commands you can enter using the Command-line input view or using the command-line debugger.

To view a list of commands, enter help all. See the MetaWare Debugger User's Guide for more details and examples.

Command
Description
addmap
Add information to a memory map
args
Display name and arguments of the current executable
attach
Attach an already running process
aux
Display ARC auxiliary registers
break
Set a breakpoint or display breakpoints
call
Call a function
continue
Start or continue execution of current process
delete break
Delete a breakpoint
delete mod
Delete an executable module
delete watch
Delete a watchpoint
delmap
Delete an entry from a memory map
detach
Detach current process from the debugger
disable break
Disable a breakpoint
disable watch
Disable a watchpoint
disassemble
Show a disassembly listing of the current process
display
Show named command-line display
download
Load an executable and read its symbol table
emem
Enter values into memory
enable break
Enable a breakpoint
enable watch
Enable a watchpoint
endm
Terminate a macro definition
evalq
Evaluate or change the value of an expression quietly
evaluate
Evaluate or change the value of an expression
exec
Execute a command when breakpoint encountered
exit
Exit the debugger
file
Display name and arguments of the current executable
file2mem
Write a file to memory
fillmem
Fill memory
focus
Set CMPD focus
globals
Display global variables
go
Stop execution at a specified address
help
Get help on a specific topic
history
Display command history
if
Execute a command if condition is true
if ... endif
Nest conditional commands
isi
Step through machine instructions, into functions
iso
Step through machine instructions, over functions
kill
Terminate the current process
lbreak
Set a breakpoint on a symbol that has not been loaded yet
load
Load a program into the debugger
locals
Display local variables
log
Log commands and results to a file
macro
Declare a macro
mem2file
Dump memory to a binary file
mem2mem
Copy memory from one location to another
memmap
Define or print memory map
memory
Dump memory
memsb
Search backward through memory
memsearch
Search forward through memory
memset
Set memory to value
mend
Terminate a macro definition
modules
List current executable and shared libraries
off
Turn off a debugger or extension toggle
on
Turn on a debugger or extension toggle
onexit command
Execute command on exiting the debugger
plot
Send commands to ARCPlotter.
print
Print a message
prop
Set a target system property
quit
Exit the debugger
read
Automatically process commands in a file
register
Display machine registers
restart
Restart the current running process
return
Return from the current function
run
Start or continue execution of the current process
set
Set or display debugger settings
show log
Show all commands and outputs thus far
simulate
Update status quietly
sleep
Sleep specified number of milliseconds
source
Display source code
ssi
Step through source-code statements, into functions
sso
Step through source-code statements, over functions
stack
Display the run-time function-call stack
status
Update status of application being debugged
stop
Stop the current process
symbols
Load symbols from executable
sysclone
Clone the current system
sysprop
Set properties of specified target system
system
Execute a command on the host system
tbreak
Set a temporary breakpoint
thread
Switch to an alternate thread
threads
Display a list of thread IDs
trace
Show instruction trace history
unmacro
Remove a macro definition
unset
Restore debugger setting to its default
variable
Declare a debugger variable
watch
Set a watchpoint or display watchpoints
watchreg
Set a watchpoint on a register
while
Execute a command while condition is true
