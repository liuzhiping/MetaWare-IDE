Interrupt state Window

The Interrupt state window displays each bit in the interrupt registers in a tree view.

Here you can see the registers included in the build and whether ARC EM features are enabled or disabled, whether each interrupt is enabled, pulse-sensitive, has any interrupts pending, and so on. The meaning of each bit is explained in the ARCv2 ISA Programmer's Reference, available where you download the ARC EM IP library.
