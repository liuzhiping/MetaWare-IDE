Working with the Registers Window

  • The Registers window displays the registers in a table.
  • The registers displayed depend on your compilation options.
  • Additional registers might appear for activities such as profiling and cycle-related simulation.
  • Values are highlighted when they change.
4.11.1.1 Changing a Value

To change the value of a register, double-click a cell and type the new value.

4.11.1.2 Changing Display Parameters

You can use the drop-down menus to change the display parameters:

  • Type a register name or range and press Enter to restrict the view.
You can type a regular expression in the field. For example, entering r1* displays all registers beginning with r1.
  • Click the down arrow and select a display format (base numbering system).
  • Click the down arrow and select which bank of registers to display (core or auxiliary).
