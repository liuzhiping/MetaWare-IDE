ARC Build-Configuration-Registers Window

When debugging on ARC hardware, you can open a Build configuration window in the Debug perspective by selecting Debugger > Build configuration when your application is running in the debugger. The window displays the contents of the build-configuration registers and explains the meanings of the bit fields.

For hardware debugging, select Hardware as the target on the Debugger tab when setting up a debug launch configuration. Then select the hardware connection, parallel-port options, and any DLL interface.
