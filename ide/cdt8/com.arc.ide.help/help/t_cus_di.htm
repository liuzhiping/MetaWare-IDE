Customizing the Disassembly Window

You can use the buttons in the Disassembly window to do the following:

  • Set or remove a breakpoint on the currently selected line.
  • Include or exclude C/C++ source intermixed with the disassembly
  • Include or exclude hexadecimal instruction bytes
  • Show hexadecimal or symbolic labels
  • Show the source of the program counter
4.8.2.1 Viewing a Specific Address

To view a specific disassembled address, specify the address in the Goto: field. Type a hexadecimal constant (for example, 0x50000), or an expression that evaluates to an address. An expression can include symbolic labels or register names (for example, typing memcmp displays the disassembled code for the memcmp function).

4.8.2.2 Profiling

To profile in this window, right-click and select Profiling.
