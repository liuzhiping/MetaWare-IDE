Debugger Windows and Views

Debugger Options Tree View

The Debugger tab of the Debug Configurations dialog and the Configure process dialog for CMPD both display the same tree view for configuring the debugger.

Select an item in the tree view to configure the related feature. The rest of the dialog changes to reflect your selection.

The items listed vary by target processor and configuration.

The Debugger Options area lists the related command-line options.
