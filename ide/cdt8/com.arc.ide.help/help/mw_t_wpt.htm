Working with Watchpoints

You can set watchpoints and manage the current watchpoints using the debugger-supplied Watchpoints window. Select Debugger > Watchpoints after your application is running in the debugger.

  • Setting Watchpoints

4.9.3.1 Setting Watchpoints

To set a watchpoint in the Watchpoints window, click the Set Watchpoint button:

The Create Watchpoint dialog opens.

  1. Use the radio buttons in the Kind area to select what you want to set the watchpoint on:
    • Expression
    • Address
    • Register
  2. Use the combobox to enter the specific address, register, or expression you want to set the watchpoint on. The interface changes based on the Kind of watchpoint selected; for an address watchpoint, you can also select the Size (number of bytes starting at the hexadecimal address specified).
Previous entries are stored in a drop-down list.
  1. In the Mode area, select write, read, or read/write.
  2. To make the watchpoint conditional, select Conditional and enter a conditional expression in the Expression: input field (for permitted expressions, see Debugger Conventions for Regular Expressions).
  3. Click OK.

4.9.3.2 Viewing Watchpoints

Each line in the listing shows the following information for a particular watchpoint:

  • Whether the watchpoint is a hardware ('H') watchpoint.
  • Whether the watchpoint is disabled ('-').
  • The name of the watched variable or register.
  • The value of the watched variable or register. In the case of an aggregate data structure, such as an array or a struct, the value of each element is displayed; for example, X [1,3,5,7,6,2,4,8]
If the value of the variable has changed, the previous value (followed by => and the current value) appears to the right of the variable name. Changed values are highlighted in red.
4.9.3.3 Managing Watchpoints

You can manage watchpoints using the Watchpoints window toolbar.

You can use the buttons on the toolbar to do the following:

  • Click to remove the currently selected breakpoint.
  • Click to remove all breakpoints shown.
  • Click to enable or disable the breakpoint on the currently selected line.
  • Click to enable or disable all breakpoints on the currently selected line.
  • Click to set a watchpoint.
  • Click to open an Examine window for the register or variable on the currently selected line.
