Using a Terminal Simulator

The MetaWare IDE provides an Eclipse terminal plugin for the MetaWare debugger to emulate a terminal being driven by a UART simulator.

After you specify a terminal simulator as described in Specifying a Terminal or COMM-Port Simulator, the Terminal window is available as soon as you launch your debug configuration.

The Terminal window is provided only for UART simulation. It is automatically created and connected when you launch a debug session with a terminal simulator specified. The launch does the connecting and the debugger does the disconnecting. You cannot connect or make settings using the buttons in the window.
