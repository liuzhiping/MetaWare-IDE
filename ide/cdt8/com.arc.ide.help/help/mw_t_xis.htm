Configuring the ARC xISS Fast Instruction-Set Simulator

The MetaWare debugger treats the ARC Fast Instruction Set Simulator (xISS) as a debug target. You can configure xISS using the Debugger Options tree view:

  1. In the C/C++ Projects view, select a project.
  2. Open the drop-down menu next to the Debug button and select Debug Configurations....
  3. Click the Debugger tab.
  4. Ensure that MetaWare Debugger is selected in the drop-down menu.
  5. In the Debugger Options tree view, select Target Selection.
  6. Select Fast ISS (xISS) from the Select the target drop-down list:
  7. Click OK.

You can now configure xISS options in the dialogs associated with other items in the tree view, or click Debug to start debugging.
