Viewing and Modifying Auxiliary Registers

You can view information about auxiliary registers in the Auxiliary registers window. Values that have changed are highlighted as you step through your program.

To open an Auxiliary registers window, launch your debug configuration and select Debugger > Aux registers.

Changing Display Parameters

You can use the window controls to change the display:

  • Starting point. Enter an auxiliary-register number from which to begin the display, then press Enter. Use a decimal number without "aux."
  • Base. Choose the numeric base in which to display the register values.
  • Registers per line
  • Size. Choose the data type in which to display the register values.
  • Toggle display of rows by increasing or decreasing offsets (addresses).
  • Change the value in an auxiliary register (select the register first).
  • Toggle display of columns by increasing or decreasing offsets (addresses).
  • Refresh the display
Changing the Value of an Auxiliary Register

To change the value of a register:

  1. Click the register to select it.
  2. Click the Change value button or right-click a cell and select Change.
  3. Enter the new value and click OK.
Not all register values can be changed.
