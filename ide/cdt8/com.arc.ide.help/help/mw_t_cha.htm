Changing a Value in a Debugger Window

Several of the windows available from the Debugger menu let you change the values of memory or variables by clicking the button to open a Change dialog.

To change a value in the Change dialog:

  1. Place your cursor in the dialog and change the value shown.
  2. Click OK.
