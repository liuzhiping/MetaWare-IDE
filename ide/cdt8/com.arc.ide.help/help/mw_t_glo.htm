Working with Global Variables

After your application is running in the debugger, you can display global variables in two different ways:

  • Select Window > Show View to open the Eclipse Variables view.
  • Select Debugger > Global Variables to open the debugger Global Variables window.

This topic tells how to use the debugger-supplied Global Variables window. For information about the Eclipse Variables view, see the Eclipse C/C++ Development User Guide.

4.14.1.1 Changing the Display
  • To add a variable to an Expressions window, select it and click .
  • To show or hide data types, click on the toolbar.
  • To show or hide addresses, click &.
  • To toggle between displaying all global variables and only those compiled with debug information, click .
  • To display a specific global variable, enter the variable name in the Name: field and press Enter.
To return to viewing all global variables, delete the text in the Name: field and press Enter.
  • To toggle alphabetical listing of variables, click .
4.14.1.2 Working with a Variable in the Global Variables Window

You can use the Global Variables window to select a variable, then perform the following debugging operations on it:

  • Click to add the variable to an Examine window.
  • Click to set a watchpoint on the variable.
  • Click to change the value of the variable.
  • Click to open a Source window for the source code containing the variable.

To save the data to a file, right click and select Save to File.
