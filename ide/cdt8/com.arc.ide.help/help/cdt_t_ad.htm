Adding Breakpoints

You can set a breakpoint on an executable line of a program. If the breakpoint is enabled when you debug, execution suspends before that line of code executes.

You can add a breakpoint in several ways:

  • In the Eclipse C/C++ Editor view, right-click the marker bar in the left margin where you want to add a breakpoint and then select Toggle Breakpoint or Toggle Hardware Breakpoint.
  • In the Debugger > Source window, double-click anywhere on the line of code where you want to add a breakpoint.
  • In the Debugger > Disassembly window, double-click anywhere on the line of code where you want to add a breakpoint.
  • In the Outline view, right-click a function and select Toggle Breakpoint or Toggle Hardware Breakpoint. The breakpoint is set on entry to the function.

In all cases, an indicator appears next to the line where you set the breakpoint. The indicator for hardware breakpoints includes an 'H.'
