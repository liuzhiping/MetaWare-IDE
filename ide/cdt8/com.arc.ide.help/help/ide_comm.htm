IDE Command-Line Options

When you launch the IDE on the command line by typing mide, you can specify the options listed in this section.

Example 
mide -noSplash
Supported Options
-build {project_name | all} - Build project from the command line

See Building Projects from the Command Line.

-cleanBuild {project_name | all} - Clean, then build project from the command line

See Building Projects from the Command Line.

-console - Launch a console window to view build process

See Building Projects from the Command Line.

-data workspace_path - Specify the path to the workspace

See Building Projects from the Command Line.

-import [uri:/]/path/to/project - Import a project to build

See Building Projects from the Command Line.

-launch configName - Launch debug configuration at startup

See Launching a Debug Configuration at Startup.

-max_stack_depth=N - Reduce debugger crashes related to Call Stack window

See Troubleshooting.

-noSplash - Launch without splash screen
