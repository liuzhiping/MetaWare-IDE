Opening the PMU Window

Select Debugger > Hardware > Power Management Unit after the PMU is enabled and your application is running in the Debug perspective.

The Power Management Unit window opens.

  • Right-click to change a register value or save the window contents to a file.
  • Expand nodes in the tree view down to the bit level.

For information on the register names and contents, see your hardware documentation.
