Waking up the Processor

To wake up (power up) the processor, do the following:

  1. Right-click the PD1 bit in the PMU_Status register in the Power Management Unit window
  2. Select Change from the pop-up menu.
  3. Type 0 in the Change value dialog and press Enter or click OK.

When you power up or power down the processor, a confirmation message appears in the Console view: 

ARC is powered down.
(off=pmu_message removes this message)

To prevent this message from appearing, specify -off=pmu_message in Specify Command-Line options text box in the Debug Configurations dialog, Command-Line Options page.
