Project Configurations

A configuration is a collection of settings for building a project. Each project includes two configurations by default, Debug and Release, which you can specify when creating a project or in Project > Properties > C/C++ Build. For example, the Debug configuration usually has different settings from the Release configuration in Project > Properties > C/C++ Build > Settings > MetaWare ARC C/C++ Compiler > Optimization/Debugging. Thus you can do all your development in the Debug configuration and switch to the Release configuration to build the final version.
