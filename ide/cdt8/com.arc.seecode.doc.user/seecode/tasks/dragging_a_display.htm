Moving tabbed displays

Each internal window can hold multiple debugger data windows and shows them as tabs in a single window. Click the tabs to switch the current display.

To move a tab from one window to another:

  1. Click the tab to bring it to focus
  2. Click the tab and drag it to the window you want to place it inside
  3. Release the mouse button over the target window to drop the tab in that window

The animation below shows clicking the Local Variables tab and placing the Locals display in a window containing Source and Disassembly.

MDI and Sash-pane only: You can move a window outside the desktop by right-clicking within the window and choosing Detach Frame from the popup menu that appears.
