Using the Cycle-Estimating Simulator (ARC)

The ARC Cycle-Estimating Simulator (CES) is a component of the ARC ISS that estimates machine cycles in executing programs. The CES also includes a cache thrash analysis routine to point out cache thrashing.

CES supports ARCtangent-A4, ARCtangent-A5, and ARC 600, and instruction and data caches. Cycle estimation is not available for code fetch where no icache is present.

To enable cycle estimation, check the box in Simulator Options. For more information on this feature, see the SeeCode Debugger User's Guide.
