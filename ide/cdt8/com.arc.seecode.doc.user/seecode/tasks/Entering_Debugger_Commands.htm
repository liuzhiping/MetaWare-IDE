Entering debugger commands

The MetaWare debugger has a built-in command-line input field where you can enter commands to control execution and invoke macros.

The command-line input field is located at the bottom of the debugger window. The text field remembers the previous commands entered. You can repeat a command by selecting it from the drop-down box, optionally edit it, and then press Enter.

Viewing a list of commands

You can use the Debug Console or Command Log to view a history of commands that have been entered. To obtain a list of commands, do any of the following:

  • View a summary and description of debugger commands in the online help.
  • Type help all in the Command: field to view a list and description of all debugger commands.
  • Type help summary in the Command: field to view a summary list of commands.
  • Type help [topic] in the Command: field to view help on a specific topic, for example help break will tell you how to set a breakpoint using the command language.
  • See the MetaWare Debugger User's Guide for a complete list of commands.
