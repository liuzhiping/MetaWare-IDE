Source window

Note:  Source is displayed only for code that was compiled with debugging information (the -g option).

The Source window lets you:

  • set or remove breakpoints
  • evaluate variables
  • view profiling information
  • save to a file

Setting or removing a breakpoint

To set or remove a breakpoint, do any one of the following in the Source window:

  • Double-click a line of source.
  • Right-click a line of source, and then click Break.
  • Select a line of source, and then click the Break button.

When the left margin shows:

  • a green arrow, the program has stopped on the line
  • a magenta arrow, a stack frame is not currently active; the arrow shows where the program will be when control re-enters the stack frame
  • a solid red rectangle, a breakpoint is set on the line
  • a grey rectangle, the breakpoint is disabled

Evaluating the value of a variable

To evaluate the value of a variable: Right-click on or within one character of a variable, and then select Evaluate.

Viewing profiling information

If the code was compiled with profiling information and your target supports profiling, you can view profiling data in the Source window.

  1. Right-click, and then select Profiling.
  2. Select any number of menu items from the top half of the menu.

    3. A column is added to the Source window that contains a profile for the type of profiling you chose.

    Tip:  Reselecting the item removes the column.

Saving your data to a file

To save your data to a file, right click and select Save to File.

Troubleshooting the Source window

Problem: Source window is not showing source code
If the Source window displays "Not stopped in known source", your source file may have been moved, renamed, or deleted from the directory in which it was compiled.
Solution:
In the Command: field, type:
set source_path=path, where path is the full path to the source file.
Example:
set source_path=c:\hcarc\demos\
Problem: Unable to set a breakpoint
To set a breakpoint, you must first compile with debugging information (the -g option).
