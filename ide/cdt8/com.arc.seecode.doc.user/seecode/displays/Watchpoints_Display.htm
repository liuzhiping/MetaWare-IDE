Watchpoints window

The Watchpoints window shows all set watchpoints. Use the window to monitor, manage, and control watchpoints.

The letter H in the left margin indicates a hardware watchpoint. Hardware watchpoints allow programs to run at full speed--program execution stops when access to the watched entity occurs. If you are stepping through source or disassembly and the program stops execution, the debugger shows if there has been any change in value after the current step is completed.

If your target system does not support hardware watchpoints, your program will stop on a watchpoint only if you are running in animation mode. Typically, processors that offer hardware assist do so only for memory, and in sizes of 1, 2, and 4 bytes. However, simulators such as the ISS offer more "hardware" assist, such as register watchpoints. Without hardware assist, the only way to run the program and still detect watchpoint changes is by animating a step action. When the debugger detects that a watchpoint has changed, it stops animation and displays the previous and changed values separated by an arrow (=>).

For example:

H register r3 [hit:4 times] data-write
value: 0x01c9b4 => 0x0384a2

Adding and removing watchpoints

To add a watchpoint:

  1. Click the Set button.
    Or
    Right-click and then select Set.
  2. Use the Set Watchpoint dialog to set the breakpoint.

To remove a watchpoint:

  1. Select the watchpoint.
  2. Click the Remove button.
    Or
    Right-click and then select Remove.

To remove all watchpoints:

  1. Click the All button next to the Remove button.
    Or
    Right-click and then select Remove All.

Enabling and disabling watchpoints

Disabled watchpoints do not halt execution or animation. You can disable a watchpoint or re-enable a disabled watchpoint anytime after the watchpoint has been added.

To disable a breakpoint:

  1. Select the watchpoint.
  2. Click the Enable/disable button.
    Or
    Right-click and then select Enable/disable.

    3. When '-' is displayed in the left margin, the watchpoint on that line has been disabled.

To disable all watchpoints:

  1. Click the All button beside the Enable/Disable button.
    Or
    Right-click and then select Enable/Disable All.
