Memory Management Unit (ARC 700)

To enable the ARC 700 Memory Management Unit (MMU) for hardware or simulator, choose Tools | Target-specific Options and check Enable MMU. You can view MMU contents and registers by opening the corresponding window under Display | Hardware.

Toggling between types of content

Click the Content: button to toggle between the following types of content:

  • all - all MMU contents
  • NZ - only sets with one or more non-zero entries
  • valid - only sets with one or more valid entries
