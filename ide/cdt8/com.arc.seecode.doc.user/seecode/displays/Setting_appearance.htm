Setting general appearance options

Use the options in the Appearance tab to set options available to alter the general look of the debugger GUI. You can change the font size, the toolbar button style, and the overall look and feel.

Opening the appearance editor GUI

To access the appearance settings select Tools | Debugger Preferences and click the Appearance tab.

Changing the font size

Changing the font size here will result in all fonts of the debugger being resized. To modify the font size:
  1. Open the Appearance editor
  2. Select a font size from the Font Size list.
  3. Click OK or Apply to commit the change.

Changing the toolbar style

To save space on toolbars, you can set them to display buttons whith only text or only icon. To modify the toolbar button style:
  1. Open the Appearance editor
  2. Select a button from the Toolbar Buttons group.
  3. Click OK or Apply to commit the change.

Changing the Look and Feel

The Look and Feel refers to the overall appearance of the GUI. By default, the debugger appears with the Look and Feel most appropriate for the host platform. To change the Look and Feel:
  1. Open the Appearance editor
  2. Select a button from the Look-and-feel button group.
  3. Click OK or Apply to commit the change.
