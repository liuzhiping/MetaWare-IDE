Debug Console

The Debug Console displays the command output from the debugger. You can use the Debug Console to view output from the debugger. You can scroll through a running list of commands and view output. The Debug Console displays the following information:

  • A list of commands that have been executed with their command-line syntax
  • A breakpoint or watchpoint that has been set or reached
  • Errors that are encountered
  • Current location of program counter and status
  • Current thread in case of a multi-threaded application
  • If the program has ended

By default, the Debug Console is docked at the bottom of the screen. To add or remove the Debug Console, select Windows | Debug Console.

For more information on debugger commands, see the Command Reference, and Entering Debugger Commands.
