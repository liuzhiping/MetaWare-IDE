Tabs

Each panel or window can contain multiple debugger displays and provides access to them using tabs. Tabs are only shown if more than one display is present in a panel or window. You can move a display from one panel or window to another using drag-and-drop.

See Setting Desktop Preferences for more information on the multiple desktop environments available.

Moving a display using its tab

To move a display:
  1. Left-click on the tab, do not release the mouse button yet.
  2. Drag the tab to the desired window or location. You can add it as another tab, or split a panel horizontally or vertically. One of the following icons will appear in place of your mouse cursor where you drag the tab:

    Move the mouse cursor to the center of a panel or window. The display will appear tabbed inside the panel or window.
    You cannot drop the display. Releasing the left mouse button will result in no action.
    Move the mouse cursor to the top of the panel or window. The display will create a new horizontal split above the panel.
    Move the mouse cursor to the bottom of the panel or window. The display will create a new horizontal split below the panel.
    Move the mouse cursor to the right edge of the panel or window. The display will create a new vertical split on the right side of the panel.
    Move the mouse cursor to the left edge of the panel or window. The display will create a new vertical split on the left side of the panel.

  3. "Drop" the display into the new position by releasing the left mouse button.
You can do the same operation by dragging-and-dropping the title bar of a panel (Sash-panel GUI only).
