Source files window

The Source Files window shows all source files for which the debugger has line information. Files are sorted alphabetically.

Note:  Regular expressions such as *.h, *.cc, or *input* can be used to restrict the number of files in the list.

The Source Files window also lists the names of all functions and/or file-scope variables that are declared in a source file: clicking '+' in the left margin expands the list.

Viewing source for a file, function, or variable

To view source for a file, function, or variable: Select a line and click the Source button.

Double-clicking a line has the following effect:

  • If the line shows a source file, a Source window opens on that file
  • If the line shows a variable, the variable is shown in an Examine window
  • If the line shows a function, a breakpoint is set at the function's entry point
