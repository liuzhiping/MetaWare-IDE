Main menu bar

The main menu provides access to the features of the debugger. The following is a listing of the menu items and links to related help topics where appropriate.

File

Restart Restarts the program and resets the target. Same function as Restart toolbar button.
New Process or Project Opens the Debug a process or processes dialog to open a different program in a new process.
Clone Desktop Creates a new debugger frame for which you can open windows. This feature is helpful for debugging programs with multiple tasks/threads or processes.
Clone Desktop and its Frames Creates a new debugger frame for which you can open windows. This feature is helpful for debugging programs with multiple tasks/threads or processes.
Print Print the contents of the window currently in focus.
Print Preview Show a preview of printed output from the window currently in focus.
Exit Debugger Shut down all processes and close the debugger.

Display

Registers Opens the Registers window.
Aux registers Opens the Auxiliary registers window. (target dependent)
Disassembly Opens the Disassembly window.
Instruction history Opens the Instruction history window.
Call Stack Opens the Call stack window.
Hardware stack Opens the Hardware stack window.
Globals Opens the Global variables window.
Functions Opens the Functions window.
Locals Opens the Local variables window.
Source Opens the Source window.
Source files Opens the Source files window.
Breakpoints Opens the Breakpoints window.
Watchpoints Opens the Watchpoints window.
Memory Opens the Memory window.
Examine Opens the Examine window.
Modules Opens the Modules window.
Profiling Opens the Profiling window.
Program output Opens the Program output window.
Log Opens the Log window.

Tools

Set Breakpoint Opens the Set Breakpoint dialog.
Set Watchpoint Opens the Set Watchpoint dialog.
Search Memory Opens the Memory Search dialog.

File-Memory-Fill Operation

 Opens the File/Memory/Fill operation dialog.
Stepping Toolbar Opens a new Stepping toolbar in a separate floating window.
Window Toolbar Opens a toolbar that contains a button for each window type. The toolbar appears as a floating window.
Debugger Options Opens the Debugger Options dialog.
Target-Specific Options Opens the Target Options dialog.
Multi-Process Debug Options Opens the CMPD Setup dialog. This only applies to multi-process debug sessions. The menu item is disabled in single-process debugging.
Edit Projects Opens the Edit Projects dialog.
Debugger Preferences Opens the Debugger Preferences dialog.

Windows

Debug Console Opens the Console.
Show All Restore all minimized debugger windows.
Hide All Minimize all debugger windows.
Refresh All Refresh content in all open and visible windows.
Cascade Order windows from largest to smallest.
Tile Horizontal Fit windows by stretching them horizontally.
Tile Vertical Fit windows by stretching them vertically.
Open Window Layout Browse the file system for a debugger window layout file (.windows).
Open Recent Window Layout Provides a list of the most recently used window layouts. Select a window layout from the list and the debugger will update its appearance and open windows based on the file. The current window layout is shown with a check mark.
Save Window Layout Save the current window layout.
Save Window Layout As Save the current window layout to a new name or location.

Dynamic menu

The dynamic menu appears between the Windows and Help menus. The dynamic menu shows the menu items of the popup menu you can obtain by right-clicking on the window currently in focus.

Help

Help The online help system opens in a new window.
Online Documentation Opens the contents file for the toolset user guides in PDF format.
Debugger Readme Opens the seecode.txt file in a text editor or the Debug Console.
About Opens the about box containing the version information. Click anywhere on the about window to close.
