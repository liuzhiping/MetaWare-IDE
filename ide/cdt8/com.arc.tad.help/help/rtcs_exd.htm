RTCS Examine TCP Control Block Window

The RTCS Examine TCP Control Block window displays the current state of a TCP control block.

Displayed Fields

TCP Control Block - State

Current state of the TCP control block, one of the following:

  • Bound
  • Close Wait
  • Closed
  • Closing
  • Established
  • FinWait 1
  • FinWait 2
  • Last Ack
  • Listen
  • Syn Received
  • Syn Sent
  • Time Wait

TCP Control Block-Last Error

Last error code received from the lower layer or the last soft error.

Local End Point

Local host IP address and port number.

Remote End Point

Remote host IP address and port number.

Type of Service

IP type of service.

Displayed fields-Receive Parameters

Window size

Size of the receive buffer.

Rx Buffer Address

Address of the default receive buffer.

Seq# first byte

Sequence number for the first byte in the receive buffer.

Seq# to receive

Sequence number for the next byte received.

Window used

Size of the window that was used.

Window Remaining

Size of the window remaining.

Max Segment Size

Maximum segment size for receiving.

Pending Rcvs

Number of pending receive commands for the connection.

Displayed fields-Send Parameters

Window Size

Size of the send buffer.

Tx Buffer Address

Address of the default send buffer.

Next Available Byte

Next available byte in the send buffer.

First Seq#

Sequence number for the first byte in the send buffer.

ReTx Timeout

Current value of the retransmission timeout.

Max ReTx Timeout

Current value of the maximum retransmission timeout.

Round Trip Time

Average estimated round-trip time.

Max Segment Size

Maximum segment size for sending.

Pending Sends

Number of pending send commands.

Size of Send Buffers

Total size of send buffers.
