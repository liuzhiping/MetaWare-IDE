RTCS TCP Statistics Window

The RTCS TCP Statistics window displays the statistics that RTCS collected for TCP.

Displayed fields

[R/T]X_TOTAL

Total number of packets sent or received.

[R/T]X_MISSED

Number of packets that were discarded due to lack of resources.

[R/T]X_DISCARDED

Number of packets that were discarded for all other reasons.

[R/T]X_ERRORS

Number of times that an internal error occurred. Internal errors indicate a catastrophic failure, and the Rx Errors and Tx Errors fields record the nature of the failure.

Field Value
Meaning
Error
RTCS error code that indicates the type of failure
Parameter
Parameter that might help you debug the failure (parameter type or value depends on the error that occurred)
Task Id
Task ID of the task that was active when the error occurred
TaskErrCode
Task error code of the active task
MemPtr
Current highwater mark for core memory
Stack
If non-zero, indicates that the stack of the active task was out of bounds when the error occurred

Rx, Dest port zero

Number of received segments with a destination port of 0.

Rx, Invalid Checksum

Number of received segments with an invalid checksum.

Rx, Invalid Options

Number of received segments with invalid options.

Rx, Invalid Source

Number of received segments with an invalid source address.

Rx, Header Too Small

Number of received segments with a header size smaller than the minimum.

Rx, Larger Than Frame

Number of received segments that are larger than the internal frames used to receive data.

Rx, Ack for Unsent Data

Number of ACKs received for data that was not sent.

Rx, Data Outside Window

Number of ACKs received for data that is outside the current buffer.

Rx, Data After Close

Number of data segments received after the connection was closed.

Rx, Segments with MSS set

Number of received segments with the maximum segment size (MSS) option set.

Rx, Segments with other options

Number of received segments with options other than MSS set.

Data Segments Received

Total number of data segments received.

Duplicate Data Received

Number of data segments received with duplicated data.

Acks Received

Number of acknowledgements received.

Duplicate Acks Received

Number of duplicate acknowledgements received.

Rx, Reset Segments

Number of reset segments received.

Rx, Window Probes

Number of probes received.

Rx, Window Updates

Number of window updates received.

SYN Not Received

Number of SYN segments expected, but not received.

ACK Not Received

Number of ACK segments expected, but not received.

Rx, SYN Not Expected

Number of SYN segments received, but not expected.

Rx, Multicast Packets

Number of multicast segments received.

Tx, Data Segment

Total number of data segments sent.

Data Segment Retransmit

Number of data segments retransmitted.

Tx, Ack-only Segment

Number of ACK-only segments sent.

Tx, Delayed Ack

Number of delayed Acks sent.

Tx, Reset Segment

Number of reset segments sent.

Tx, Window Probe

Number of window probes sent.

Tx, Window Update

Number of window updates sent.

Active Opens

Number of active open requests.

Passive Opens

Number of passive open requests.

Established Connections

Total number of established connections.

Graceful Shutdowns

Number of graceful shutdown operations.

Connections Aborted

Number of aborted connections.

Failed Connections

Number of failed connections.

Aborts

Total number of abort requests received.
