I/O Devices Window

The I/O Devices window provides a list of the I/O devices that an application has dynamically installed.

Displayed Fields

The following information is displayed for each I/O device:

  • string identifier that identifies the device
  • open function used to initialize the device
  • close function used to uninstall the device
  • read function used to read from the device
  • write function used to write to the device
  • I/O control function used to process I/O control commands for the device

Functions are shown by name, with their addresses in brackets. Only the first 13 characters of names are displayed.
