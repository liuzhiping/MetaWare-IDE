RTCS UDP Configuration Window

The RTCS UDP Configuration window displays the configuration for UDP.

Displayed Fields

UDP Last Port Used

Next port number that is returned when an open request is received with the local port set to 0.

UDP Control Block Summary

List of current UDP control blocks (UCBs).

For each UDP socket, RTCS allocates a UCB, which maintains the relevant UDP data for the connection.

UCB

Address of the UCB.

Local IP Address

Local IP address of the UCB.

Local Port

Local UDP port number being serviced by the UCB.

Remote IP Address

Remote IP address of the UCB.

Remote Port

Remote UDP port number being serviced by the UCB.

Incoming Packets

Number of incoming packets that are waiting to be received on the port.
