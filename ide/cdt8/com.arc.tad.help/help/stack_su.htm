Stack Summary window

stackThe Stack Summary window contains a list of the stack usage of the tasks and the interrupt stack.

Displayed fields

Task-the name of the task.

Stack Base-the base address of the associated stack for the task. For an ARC processor, this address is the top of the stack.

Stack Limit-the end of the stack. For an ARC processor, this address is the bottom of the stack.

Stack Used-the lowest address currently in use in the stack. For a suspended task, this value is the stack pointer minus the number of bytes to store the task context information.

Overflow-whether a stack overflow condition occurred.

Percentage-percentage of the stack that is in use.
