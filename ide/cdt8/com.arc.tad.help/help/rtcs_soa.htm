RTCS Socket Summary Window

The RTCS Socket Summary window displays a list of the current sockets that have been created using RTCS.

Displayed Fields

The information displayed for each socket is:

  • socket handle (value returned to application when is calls socket())
  • current state of the socket:
    • Abort
    • Bound
    • Closing
    • Connected
    • Connecting
    • Listening
    • Unbound
  • socket type, either Stream or Datagram
  • callback function as specified by the application
  • current number of owners of the socket

Controls

Examine a Socket

To view a socket in more detail and open the RTCS Examine Socket window for that specific socket, double-click the socket.
