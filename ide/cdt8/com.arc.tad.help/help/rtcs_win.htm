RTCS Windows

RTCS ARP Statistics Window

The RTCS ARP Statistics window displays the statistics that RTCS gathered for ARP.

Displayed fields

IP Interface

IP address and net mask being serviced by ARP.

[R/T]X_TOTAL

Total number of packets received or sent.

[R/T]X_MISSED

Number of packets that were discarded due to lack of resources.

[R/T]X_DISCARDED

Number of packets that were discarded for all other reasons.

[R/T]X_ERRORS

Number of times that an internal error occurred. Internal errors indicate a catastrophic failure, and the Rx Errors and Tx Errors fields record the nature of the failure.

Field value
Meaning
Error
RTCS error code that indicates the type of failure
Parameter
Parameter that might help you debug the failure (parameter type or value depends on the error that occurred)
Task Id
Task ID of the task that was active when the error occurred
TaskErrCode
Task error code of the active task
MemPtr
Current highwater mark for core memory
Stack
If non-zero, indicates that the stack of the active task was out of bounds when the error occurred

Requests received

Number of valid ARP requests received.

Replies received

Number of valid ARP replies received.

Requests sent

Number of ARP requests sent.

Replies sent

Number of ARP replies sent.

ARP Alloc failed

Number of times an ARP entry could not be allocated.

Cache hits

Number of ARP cache hits.

Cache misses

Number of ARP cache misses.

Packet discards

Number of data packets that were discarded due to a missing ARP entry.
