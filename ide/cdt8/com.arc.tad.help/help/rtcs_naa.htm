RTCS NAT Statistics Window

The RTCS NAT Statistics window displays network address translation statistics.

Displayed fields

Total amount of sessions created

Total number of sessions created to date.

Number of sessions currently open

Number of sessions currently open.

Max. number of sessions simultaneously open

Maximum number of sessions open simultaneously to date.

Number of packets processed by NAT

Number of packets processed by ARC NAT.

Number of unmodified packets

Number of unmodified packets.

Number of packets from public to private

Number of packets from public to private realm.

Number of packets from public to private with error

Number of packets from public to private realm with errors (packets that have errors are discarded).

Number of packets from private to public

Number of packets from private to public realm.

Number of packets from private to public with error

Number of packets from private to public realm with errors (packets that have errors are discarded).
