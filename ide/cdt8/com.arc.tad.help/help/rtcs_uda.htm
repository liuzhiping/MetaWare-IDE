RTCS UDP Statistics Window

The RTCS UDP Statistics window displays the statistics that RTCS collected for UDP.

Displayed Fields

[R/T]X_TOTAL

Total number of packets sent or received.

[R/T]X_MISSED

Number of packets that were discarded due to lack of resources.

[R/T]X_DISCARDED

Number of packets that were discarded for all other reasons.

[R/T]X_ERRORS

Number of times that an internal error occurred. Internal errors indicate a catastrophic failure, and the Rx Errors and Tx Errors fields record the nature of the failure.

Field Value
Meaning

Error
RTCS error code that indicates the type of failure

Parameter
Parameter that might help you debug the failure (parameter type or value depends on the error that occurred)

Task Id
Task ID of the task that was active when the error occurred

TaskErrCode
Task error code of the active task

MemPtr
Current highwater mark for core memory

Stack
If non-zero, indicates that the stack of the active task was out of bounds when the error occurred

Rx, Dest Port Zero

Number of packets received with a destination port of 0.

Rx, Bad Checksum

Number of received packets with an invalid checksum.

Rx, Smaller Than Header

Number of received packets that are smaller in length than a valid header.

Rx, Larger Than Frame

Number of received packets that are larger than the internal frames used to receive data.

Rx, for a Closed Port

Number of packets received for unopened ports.
