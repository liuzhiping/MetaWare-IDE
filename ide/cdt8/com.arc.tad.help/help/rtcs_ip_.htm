RTCS IP Configuration Window

The RTCS IP Configuration window displays the configuration specific to IP.

Displayed Fields

Next Packet ID

Next packet identification number to be used.

IP Control Blocks?

Indicates whether any current IP control blocks (ICBs) exist. Any current ICBs are listed.

RTCS uses ICBs to maintain a connection between IP and the upper-layer protocols.

Field Value
Meaning

ICB
Address of the ICB

Protocol
Protocol identifier of the upper-layer protocol that the ICB services

ServiceFn
Address of the function called to service the ICB

Config Data
Parameter for the service function

Gateways Defined?

Indicates whether gateways have been defined. Any defined gateways are listed.

Field Value
Meaning

Gateway
IP address of the gateway

NetMask
IP (sub)network mask to route

Network
IP (sub)network to route

Connected Interfaces?

Indicates whether there are any connected interfaces. Connected interfaces are listed.

Field value
Meaning

NetIF
Address of the network interface structure used to maintain the interface

Address
IP address for the interface

LocMask
IP (sub)network mask for the address

NetMask
IP (sub)network mask for the interface

Controls

Connected Interfaces

Double-click one of the interfaces in the list to open the RTCS Examine Interface Connection window for the interface.
