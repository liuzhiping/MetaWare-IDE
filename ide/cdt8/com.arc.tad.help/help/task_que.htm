Task Queues Window

The Task Queues window displays the current state of a particular task queue.

Displayed Fields

This window displays the following for task queues:

  • address of the task queue being examined
  • type of queue:
    • Priority-tasks are queued in order of their priority.
    • FIFO-tasks are queued in first-in, first-out order regardless of their priority.

Waiting Tasks

All tasks waiting in the queue. Tasks are listed from the head of the queue to the end of the queue. Each entry contains the name of the task, task ID, and task descriptor.
