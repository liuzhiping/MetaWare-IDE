Lightweight Semaphores Window

The Lightweight Semaphores window shows the list of lightweight semaphores.

Displayed Fields

The information that is displayed for each lightweight semaphore is:

  • LWSem-variable name and/or address of the lightweight semaphore
  • Valid-whether the semaphore is valid
  • Value-current count (value) of the semaphore
  • Waiting#-number of tasks waiting for the semaphore

If a semaphore has waiting tasks, you can see a list by expanding the entry for the semaphore.
