Initialization Window

The Initialization window shows the MQX initialization information that the application provided.

Displayed Fields-Interrupts

Interrupt Stack Size:

Size of the interrupt stack. The value represents the maximum amount of stack space used by all interrupt handlers.

HW Interrupt Level:

Maximum hardware-interrupt priority level of MQX. An MQX-disable call disables only to this level.

Displayed Fields-Kernel Memory Bounds

Start:

Lowest address of memory for use by MQX.

End:

Highest address of memory for use by MQX. You must ensure that you have allocated sufficient memory for all the tasks.

Displayed Fields-Maximums

Msg Pools:

Maximum number of message pools (private pools and system pools) that can be created. The default maximum is 255.

Msg Queues:

Maximum queue number that can be used to open a queue. The default maximum is 255.

Displayed Fields-Default I/O

I/O Stream:

String that is associated with the default I/O stream.

I/O Mode:

Processing that is associated with the default I/O stream; for example, whether echoing is to be done.
