Partitions Window

The Partitions window shows the current state of each partition and its partition blocks.

Displayed Fields

Partition

The address of the partition.

Type

Partition type, either Dynamic or Static.

Block Size

Number of bytes in each block in the partition.

Free#

Number of currently unused blocks.

Total#

Total number of blocks in the partition.

Grow#

Number by which a dynamic partition can grow when all blocks are allocated. Grow# does not apply to static partitions.

Limit

Maximum number of blocks that can be created for a dynamic partition. Limit does not apply to static partitions.

Max Used

Maximum number of blocks allocated at any one time. The value is a highwater mark.

Partition Block List

The list contains an entry for each block in the partition. Each entry contains the address of the block and the state of the block. Blocks can either be owned by a task or free.
