What Is TAD?

TAD is a plugin to the MetaWare debugger for browsing and analyzing MQX and RTCS data structures that control the execution of an application

More specifically, TAD provides the following capabilities to the MetaWare debugger:

  • examine data structures
  • get a summary of all tasks
  • view the context of a specific task
  • traverse the ready queue
  • browse message queues and message pools
  • inspect message contents
  • browse lightweight semaphores and view waiting tasks
  • browse semaphores and view owning and waiting tasks
  • browse events and examine state
  • get a summary of and inspect the content of mutexes
  • audit task and MQX memory resources
  • view partition memory
  • examine interrupt-handling mechanisms
  • view stack usage
  • inspect MQX instrumentation logs
  • examine processor usage
  • display kernel time
  • display RTCS statistics
  • examine RTCS internal data

We assume that you are familiar with MQX and its functionality. For more information, see MQX User's Guide and MQX Reference.

We also assume that you are familiar with RTCS and its functionality. For more information, see RTCS User's Guide.
