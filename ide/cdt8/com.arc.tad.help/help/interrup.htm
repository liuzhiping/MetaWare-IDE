Interrupts Window

The Interrupts window displays a summary of interrupt handlers on the current processor.

The table contains an entry for each supported interrupt. Each entry contains the interrupt number, the address or name of the interrupt service routine (ISR), the ISR data, and the address or name of the exception handler.

Displayed Fields

Kernel ISR

Address or name of the kernel ISR (the first-level interrupt handler), usually written in assembly language.

Default ISR

Address or name of the MQX default ISR for the supported interrupts.

List Entries

Displays this information for each supported interrupt:

  • interrupt number
  • ISR address or name
  • ISR data
  • exception ISR address or name
