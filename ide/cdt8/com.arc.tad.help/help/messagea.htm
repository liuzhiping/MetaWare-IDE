Message Queues Window

The Message Queues window shows a description of each message queue and provides access to the messages in the queue as well as the owner of the queue and the notification function that is associated with the queue.

Displayed Fields-Queue Entries

Queue #

The ID number of the queue.

Entries

Current number of messages in the queue.

Type/Owner

In the case of a private queue, then the owning task is shown. In the case of a system queue, the type System Q is shown.

Displayed Fields-Queue Details

Maximum

Maximum number of entries allowed in the queue. Some queues are of unlimited size.

First Entry

Address of the first message in the queue.

Queue Type

Private Queue or System Queue.

Owner Task

For a private queue, the task descriptor of the task that owns the queue is displayed. This field is not applicable for a system queue.

Notification Function

The name of the function or the address of the function, if applicable. MQX calls the function when it puts a message in the queue.

Notification Function Parameter

The parameter passed to the notification function if a notification function is applicable.

Message List

List of messages, if any, within the queue. Each message has the following fields displayed:

Message

The address of the message.

Source Q#

The ID number of the source queue of this message.

Processor#

The processor where the message originated.

Msg Size

The size of the message in bytes.
