MQX > General Menu

Kernel Data

Open the Kernel Data window, to view general MQX information.

Memory Blocks

Open the Memory Blocks Summary window to view the blocks of memory allocated by MQX.

Partitions

Display a summary of the partitions created by the application.

Lightweight Memory Pools

Open a window to show the lightweight memory pools.

Log Summary

Display the status of the logs.

Initialization

Display initialization information used by MQX.

Interrupts

Display a summary of the application's interrupt-handling routines.

IO Devices

Display a summary of the I/O device handlers that have been installed.

Names

Display the current names in the name table and their associated values.

Lightweight Timers

Open a window to show the state of lightweight timers.

Check for Errors

Open the Check System for Errors window to display a list of detected errors or data corruption in MQX data structures.
