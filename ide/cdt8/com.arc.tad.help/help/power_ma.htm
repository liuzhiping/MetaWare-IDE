Power Management Window

The Power Management window is available with MQX-EP.

Displayed fields

PMU

Power Down Mode:  0=disabled, 1=power down CPU, 2=power down CPU and memory

Power Down Threshold: The number of consecutive times MQX returns to the idle task before enabling the power-down mode.

Maximum Slow Down Number: Number of times to lengthen the timer period  (You can lengthen the timer period to allow a longer power-down interval. You can also specify the number of times the timer interval is increased.)

Current Slow Down Number: The number of times the timer interval has been increased (from 0 to the Maximum Slow Down Number).

PMU Power Consumption:  Estimated power consumption

Current Idle Count:  Number of times the idle task has been entered (power down occurs after the Power Down Threshold is reached).

DVFS

Global DVFS Mode: System-wide DVFS mode, one of the following:

  • 0 = Full Power (default)
  • 1 = 75% Power
  • 2 = 50% Power
  • 3 = 25% Power

This setting applies to all tasks unless specifically overridden.

Automatic DVFS Mode Enabled:  Yes=On, No=Off; default No.

Power Consumption For DVFS Mode N:  Estimated power consumption

Current DVFS Mode:  Same as Global DVFS Mode.
