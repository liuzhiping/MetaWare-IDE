Troubleshooting

TAD displays `Unknown' for some fields

If TAD cannot find the file mqxutil.dll, it displays Unknown for some fields. Other than this behavior, TAD operates properly.

When TAD was installed, mqxutil.dll was copied to either your windows\system directory or your winnt\system32 directory, as is appropriate for your host environment. If the file is no longer in the appropriate directory, copy it from the distribution CD.

TAD displays `Unknown' for CPU type fields only

The CPU types in your version of mqxutil.dll do not include all the CPU types that MQX supports. Contact product support to get the latest version of mqxutil.dll.
