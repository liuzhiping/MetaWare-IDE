RTCS TCP Control Block Summary Window

The RTCS TCP Control Block Summary window displays a list of all active TCP control blocks (TCBs).

Displayed fields

The following information is displayed for each TCB:

  • address
  • state
  • last error
  • local port
  • local address
  • remote port
  • remote address

Controls

Double-click a TCB entry to open the RTCS Examine TCP Control Block window for the TCB.
