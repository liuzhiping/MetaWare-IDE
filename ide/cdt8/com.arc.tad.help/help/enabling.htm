Enabling TAD in the MetaWare Debugger

To enable task-aware debugging do either of the following:

  • specify -OS=MQX when invoking the MetaWare debugger from the command line, or
  • specify MQX as the RTOS on the Program Options page of the Debugger Options dialog or the IDE Debug Configurations Dialog:
