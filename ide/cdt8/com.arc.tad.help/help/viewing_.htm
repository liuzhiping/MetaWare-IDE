Viewing the Context of Blocked Tasks Using TAD

You can use the Task menu to view the context of a blocked task. All tasks are listed under the Task menu, and the current (active) task is indicated.

To view the context of a blocked task, select the name of the task. TAD updates the following windows to reflect values for the task that you select:

  • Call Stack window
  • Register window
  • Local Variables window
  • Source window
