RTCS NAT Configuration Window

The RTCS NAT Configuration window provides information about the configuration of a border router on a private network.

Displayed fields

IP address

Private network IP address of the ARC NAT border router.

Network Mask

Subnet mask of the private network on which the ARC NAT border router resides.

Session x

TCP/UDP or ICMP session number. Multiple sessions are supported between hosts on the public and private network.

Protocol

Protocol of the session (TCP/UDP or ICMP).

Controls

Examine a session

To examine a session using an RTCS Examine NAT Session window, double-click the session icon.

Depending on the protocol being used, the RTCS Examine NAT Session window displays either ICMP or TCP/UDP information for the created session.
