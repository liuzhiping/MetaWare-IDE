RTCS Examine Socket Window

The RTCS Examine Socket window displays the current state of a socket

Displayed Fields

Socket

Socket handle being examined.

Socket - State

State of the socket; one of the following:

  • Abort
  • Bound
  • Closing
  • Connected
  • Connecting
  • Listening
  • Unbound

Socket - Type

Type of socket, either Stream or Datagram.

Local End Point

Port number and IP address of the local endpoint. The fields are set when the socket is bound.

Remote End Point

Port number and IP address of the remote endpoint. The fields are used only for stream sockets that are in a connected state.

Options

Current values of the socket options. For information on each option, see RTCS User's Guide.

Listen Error

Error that occurred if a socket listen operation failed.

Last Error

Last error that occurred during the operation of the socket.

Outstanding Accept

If the field is True, the socket has an accept operation pending.

Outstanding Connect

If the field is True, the socket has a connect operation pending.

TCP Control Block

Address of the TCP control block for the socket. The field is used only for stream sockets.

UDP Control Block

Address of the UDP control block for the socket. The field is used only for datagram sockets.

Owners

List of the tasks that own the socket.

Controls

TCP Control Block line

Double-click to open the Examine TCP Control Block window for the stream socket.

UDP Control Block line

Double-click to open the UDP Configuration window for the datagram socket.
