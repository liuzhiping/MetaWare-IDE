RTCS Examine NAT Session Window

The RTCS Examine NAT Session window displays ICMP or TCP and/or UDP statistics, depending on the protocol in use.

To display the window for a session, double-click the session icon in the RTCS NAT Configuration window.

Displayed Fields

Session

ARC NAT session number, assigned by TAD.

Protocol

Protocol of the session (TCP/UDP or ICMP).

Private host IP

Network IP address of a host on the private network.

Public host IP

Network IP address of an external host on the public network.

Replacement IP

Network IP address that the NAT border router has provided for a host on the private network to communicate with a host on the public network.

Private host Port

TCP/UDP port number that the host on the private network is using for the session.

Public host Port

TCP/UDP port number that the external host on the public network is using for the session.

Replacement Port

TCP/UDP port number of the NAT border router.

Private ICMP ID

ICMP query ID that the host on the private network is using for this session.

Public ICMP ID

ICMP query ID that the external host on the public network is using for this session.
