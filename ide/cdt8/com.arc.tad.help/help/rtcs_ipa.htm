RTCS IP Statistics Window

The RTCS IP Statistics window displays the statistics that RTCS collected for IP.

Displayed Fields

[R/T]X_TOTAL

Total number of packets received or sent.

[R/T]X_MISSED

Number of packets that were discarded due to lack of resources.

[R/T]X_DISCARDED

Number of packets that were discarded for all other reasons.

[R/T]X_ERRORS

Number of times that an internal error occurred. Internal errors indicate a catastrophic failure, and the Rx Errors and Tx Errors fields record the nature of the failure.

Field Value
Meaning

Error
RTCS error code that indicates the type of failure

Parameter
Parameter that might help you debug the failure (parameter type or value depends on the error that occurred)

Task Id
Task ID of the task that was active when the error occurred

TaskErrCode
Task error code of the active task

MemPtr
Current highwater mark for core memory

Stack
If non-zero, indicates that the stack of the active task was out of bounds when the error occurred

Discarded, error in IP Header

Number of received packets that were discarded due to an error in the IP header.

Discarded, illegal destination

Number of received packets that were discarded because of an illegal destination.

Discarded, unrecognized protocol

Number of received packets whose protocol field did not match the protocols currently being serviced.

Packets delivered to upper layer

Number of packets that IP delivered to upper-layer protocols.

Packets forwarded

Number of IP packets that were forwarded.

Packets with incorrect version

Number of received packets with an invalid version number. These packets are included in Discarded, error in IP Header.

Packets with invalid checksum

Number of received packets with an invalid checksum. These packets are included in Discarded, error in IP Header.

Packets with invalid src addr

Number of received packets with an invalid source address. These packets are included in Discarded, error in IP Header.

Packets with header too small

Number of received packets with headers smaller than the minimum size. These packets are included in Discarded, error in IP Header.

Packets smaller than header

Number of received packets with a packet size that is smaller than the header size. These packets are included in Discarded, error in IP Header.

Packets larger than frame

Number of received packets that are larger than the internal frames used to receive data. These packets are included in Discarded, error in IP Header.

Packets with TTL=0

Number of received packets with the time to live set to 0. These packets are included in Discarded, error in IP Header.

Number of received IP fragments

Number of received IP fragments.

Number of reassembled fragments

Number of reassembled fragments.

Number of discarded fragments

Number of received fragment packets that were discarded.

Number of sent fragments

Number of sent fragment packets.

Number of packets fragmented

Number of sent IP packets that were fragmented.

Number of fragmentation failures

Total number of fragmentation failures on transmission.
