RTCS IPIF Statistics Window

The RTCS IPIF Statistics window displays the statistics that RTCS collected for the IP interface.

Displayed Fields

IP Interface

IP address and net mask being serviced by the interface.

[R/T]X_TOTAL

Total number of packets sent or received.

[R/T]X_MISSED

Number of packets discarded due to lack of resources.

[R/T]X_DISCARDED

Number of packets discarded for all other reasons.

[R/T]X_ERRORS

The number of times that an internal error occurred. Internal errors indicate a catastrophic failure, and the Rx Errors and Tx Errors fields record the nature of the failure.

Field Value
Meaning

Error
RTCS error code that indicates the type of failure

Parameter
Parameter that might help you debug the failure (parameter type or value depends on the error that occurred)

Task Id
Task ID of the task that was active when the error occurred

TaskErrCode
Task error code of the active task

MemPtr
Current highwater mark for core memory

Stack
If non-zero, indicates that the stack of the active task was out of bounds when the error occurred

Total bytes received

Total number of bytes received by the interface.

Unicast packets received

Number of unicast packets received by the interface.

Multicast packets received

Number of multicast packets received by the interface.

Broadcast packets received

Number of broadcast packets received by the interface.

Total bytes sent

Total number of bytes sent by the interface.

Unicast packets sent

Number of unicast packets sent by the interface.

Multicast packets sent

Number of multicast packets sent by the interface.

Broadcast packets sent

Number of broadcast packets sent by the interface.
