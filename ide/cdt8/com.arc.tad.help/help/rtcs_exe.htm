RTCS Examine UCB Window

The RTCS Examine UCB window displays the statistics that RTCS collected for UCB. To display the RTCS Examine UCB window, double-click the icon in the RTCS UDP Configuration window.

Displayed Fields

UCB

Address of the UCB.

Local IP Addr

Local IP address being serviced by the UCB.

Local Port

Local UDP port number being serviced by the UCB.

Remote IP Addr

Remote IP address being serviced by the UCB.

Remote Port

Remote UDP port number being serviced by the UCB.

Rx Requests

Number of receive requests from the upper layer that are waiting on the port.

Bypass Rx

When the field is True, RTCS does not verify checksums on incoming packets.

Bypass Tx

When the field is True, RTCS does not generate checksums on outgoing packets.
