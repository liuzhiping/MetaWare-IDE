RTCS Socket Configuration Window

The RTCS Socket Configuration window displays socket-layer configuration information.

Displayed Fields

Socket Initialized?

Indicates whether the socket layer is initialized.

Number of Sockets

Current number of sockets.

Select List?

Indicates whether there are current entries on the select list.

Controls

Number of Sockets field

Double-click the field to open the RTCS Socket Summary window.
