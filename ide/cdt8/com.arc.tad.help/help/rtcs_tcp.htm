RTCS TCP Configuration Window

The RTCS TCP Configuration window displays the configuration information for TCP.

Displayed Fields

TCPIP Status

Status of the TCP/IP layer. The field is set to OK unless an error occurred that makes it difficult for the TPC/IP layer to continue to operate.

Active TCP Control Blocks?

Indicates whether any current TCP control blocks exist.

For each TCP connection, RTCS allocates a TCP control block, which maintains the relevant TCP data for the connection.

Active Rx Chunk Blocks?

Indicates whether any current Tx chunk blocks exist.

Under some circumstances, RTCS must receive data into internal TCP buffers. RTCS uses Rx chunk blocks to define the internal TCP buffers.

Free Rx Chunk Blocks?

Whether any current entries exist in the free list.

When RTCS no longer needs an allocated Rx chunk block, it places the block in the free list. RTCS allocates Rx chunk blocks from the free list before it allocates them from core memory

Free Send Buffer Chains?

Indicates whether there are any send-buffer chains in the free list.

RTCS uses send-buffer chains to define the contents of transmit buffers. RTCS allocates send-buffer chains per TCP control block and puts them on the free list when it no longer needs them. RTCS allocates send-buffer chains from the free list before it allocates them from core memory.

Next port to assign

Next port number returned when an open request is received with the local port set to 0.

Timeout Queue?

Indicates whether any current entries exist in the pending timeout queue.

Expired Delayed ACK timers?

Indicates whether any delayed ACK timers expired and are awaiting to be serviced.

Expired rexmit/probe/timewait?

Indicates whether any retransmit, probe, or timewait timers expired and are waiting to be serviced.

Expired ack-of-transmission timers?

Indicates whether any transmission-acknowledgement timers expired and are waiting to be serviced.

Expired receive timeout timers?

Indicates whether any receive timers expired and are waiting to be serviced.

Pending Open Requests?

Indicates whether any open requests are pending.

Controls

Active TCP Control Blocks field

Double-click the field to open the RTCS TCP Control Block Summary window.
