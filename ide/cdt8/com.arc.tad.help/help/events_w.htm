Events Window

The Events window lists the created event groups on this processor. Each row corresponds to an event group. Columns display information pertaining to the event group.

Displayed Fields

The information displayed for each event group is:

  • Event: the address of the event group.
  • Waiting #: the number of tasks waiting for this event.
  • Bit Mask: the current bit mask (value of the event group).
  • Name: the identifier or name of the event group.

Displayed Fields

Valid?

Certain structures and components within MQX contain internal verification fields. If these fields are not set to the appropriate validation stamp, the integrity of the data displayed in the current window is suspect. A No value here might indicate that a component is not installed or internal MQX data structures are corrupted.

Events

Event groups are 32 bits. The application defines what each bit represents. A task can wait for any combination of these bits. The fields show the current setting of each of the event bits. An application can set event bits by calling _event_set() and clear them by calling _event_clear().

Task List

The list contains an entry for each task that is waiting for event bits in the event group.

The entries in the task list can be expanded to show details about the associated task. This information is identical to the information shown in the Task Summary window.
