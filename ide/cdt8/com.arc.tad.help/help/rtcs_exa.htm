RTCS Examine Interface Connection Window

The RTCS Examine Interface Connection window displays the current state of a IP interface.

Displayed Fields

Interface

Address of the structure that defines the interface connection.

Handle

Address of an interface-specific data structure.

Open Fn

Address of the function used to open or initialize the interface.

Close Fn

Address of the function used to close or uninstall the interface.

Join Fn

Address of the function used to join the interface.

Leave Fn

Address of the function used to leave the interface.

MinTxUnit

Minimum size of a transmit unit.

BootFn

Address of the boot function.

Boot

Data provided to the boot function.

ARP Cache Entries

For each cache entry, the following information is listed:

  • IP address
  • Ethernet address
  • complete resolution of the IP address
  • aged entry
  • entry hit
  • waiting-address resolution
  • resend interval
