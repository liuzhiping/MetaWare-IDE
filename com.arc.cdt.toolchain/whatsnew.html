What's New in the MetaWare® IDE

The MetaWare IDE uses modular Eclipse technology to seamlessly integrate the MetaWare® Development Toolkit from ARC™ International.

  • Use a single IDE to code, build, and debug with the MetaWare debugger
  • Compile, link, and debug in one step
  • Auto-generate makefiles
  • View MetaWare-specific debugger information
  • Cross-reference and browse C/C++ code
  • Search using intelligent-identifier search
  • Add your own plugins using extensible Java™ technology
  • Take advantage of custom help for ARC customers
  • Consolidate projects in a single workspace on your system
  • Access project files easily from your file system
  • Lower the risk of accidentally losing files with the built-in history mechanism
  • Annotate error messages, to-do lists, search hits, and debugger breakpoints using the built-in marker mechanism
  • Easily integrate revision control to track changes to workplace resources
  • Automatically rebuild on change to any file in a project

Note: The MetaWare IDE does not support the Eclipse Ant feature documented in the online Workbench User Guide.
