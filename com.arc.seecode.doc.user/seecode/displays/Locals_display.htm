Local Variables window

The Local Variables window displays the local variables of your program. All scopes from the current function's innermost scope out to the file scope are included. Non-exported (static) variables are also shown (non-exported variables are local to the current file).

The variables are listed one per line, from innermost to outermost scope. Scope indicators show you the boundaries between scopes. To show or hide addresses, click A.

Local variables are displayed only for functions that were compiled with debugging information (the -g option).

Saving your data to a file

To save your data to a file, right click and select Save to File.
