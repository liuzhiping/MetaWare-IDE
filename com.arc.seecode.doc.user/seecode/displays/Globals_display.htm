Global Variables window

The Global Variables window shows the global variables of your program. Each global variable is listed on a separate line. To show or hide addresses, click A.

The window lets you:

  • change the value of a variable
  • set a watchpoint on a variable

Changing the value of a variable

To change the value a variable:

  1. Click the variable, and then click the Change button. Or Right-click the variable, and then click Change. The Change dialog is displayed.
  2. Edit the value in the New value field of the Change dialog, and then click OK.

Setting a watchpoint

When the left margin shows:

  • a blue rectangle, one or more of the registers has a watchpoint set on that line
  • a gray rectangle, all watchpoints are disabled (execution does not stop on a disabled watchpoint)

To set a watchpoint on a variable:

  1. Click on the variable, and then click the Watch button. Or Right-click the variable, and then select Watch.

Saving your data to a file

To save your data to a file, right click and select Save to File.
