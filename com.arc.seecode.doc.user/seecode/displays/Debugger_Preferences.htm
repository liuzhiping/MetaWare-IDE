Setting preferences

Use the Debugger Preferences dialog to configure user interface settings, edit the keyboard shortcut for stepping actions, and create and edit keyboard shortcuts for debugger commands and user-defined macros. To access Debugger Preferences, select Tools | Debugger Preferences.

The configuration information set in the debugger preferences dialog persists for every project for the current user. The preferences are stored in a file located in /$userhome/.SeeCode/scprops.xml.

Use the debugger Preferences dialog to set the following properties:

  • Desktop Settings: Choose the type of desktop environment you prefer and set options specific to the desktop style.
  • Appearance: Set font size, toolbar button style, and Look & Feel.
  • Stepping Keys: Change the keyboard shortcuts for stepping actions.
  • Command and Macro Keys: Add, change, or remove keyboard shortcuts for user-defined macros and debugger commands.

Applying preferences

When you click OK or Apply, the preferences take effect immediately and it is not necessary to restart the debugger.
