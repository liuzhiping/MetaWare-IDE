Adding/removing the options toolbar

Every display has an Options toolbar. The Options toolbar provides functions for the specific display. For example, the Registers display contains buttons for Change and Watch register.

To enable the Options toolbar:

  1. Right-click in the window to add/remove the options toolbar.
  2. Select Toolbars | Options, or Toolbars | <display> options, where <display> is the name of the display, such as Source or Registers.

To remove the options toolbar, right-click and select Toolbars | Options, or Toolbars | <display> options again.

The Options toolbar will appear at the top of the window.
