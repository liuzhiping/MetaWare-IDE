Registers window

The Registers window shows rows and columns of machine registers and their values. The Registers window highlights changed values in red. The Registers window lets you:

  • set a watchpoint on a register
  • change the format of displayed values
  • modify a register value
  • restrict the display to a subset of registers
  • change the base numbering system for registers
  • change the number of registers displayed per line

Tip:  You can modify how the debugger displays registers through the Options toolbar in the Registers window.

Setting a watchpoint on a register

To set a watchpoint on a register:

  1. Click the register name, and then click the Watch button.
    Or
    Right-click the register value, and then select Watch.

When the left margin shows:

  • a solid blue rectangle, a watchpoint is set on one or more of the registers on that line
  • a grey rectangle, all watchpoints are disabled (program execution does not stop on a disabled watchpoint)

Changing the format of displayed values

To change the format of displayed values:

  1. In the Format box, select one of the following options:
    • uint (unsigned integer format)
    • int (integer format)
    • float (floating point number)
    • frac (fraction)

Modifying a register value

To modify the value a register:

  1. Click the register name, and then click the Change button.
    Or
    Right-click the register value, and then select Change.

    2. The Change dialog is displayed.

  2. Modify the value in the New value field, and then click OK.

Restricting the display to a subset of registers

To restrict the number of registers on display by register name:

  1. In the Name: field, enter a register name or a range of register identifiers, and then press ENTER.

    2. Tip:  You can type a regular expression in this field. For example, entering r1* displays all registers whose names begin with "r1".

To restrict the number of registers on display by register bank:

  1. In the Bank box, select a register.

Changing the base numbering system for registers

To change the base numbering system for registers:

  1. In the Base box, select one of the following options:
    • 16 (hexadecimal)
    • 10 (decimal)
    • 8 (octal)
    • 2 (binary)

    Note:  Your selection is ignored when register values are displayed as floating-point numbers or fractions.

Changing the number of registers displayed per line

To change the number of columns of registers to display per line:

  1. In the Registers per line box, select the number of columns to display.
