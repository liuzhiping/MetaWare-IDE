Command and status bar

Command-line

Enter command-line input into the Command: field. Press Enter to commit the command. See Entering Debugger Commands and Command Reference.

Status bar

The status bar displays messages and status events from all debugger windows.

Running/Stopped indicator

Indicator shows the following states:
  • Stopped when the debugger is idle.
  • Run # when the process is executing
  • and Animating when the process is in animation mode.
