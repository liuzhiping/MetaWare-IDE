Auxiliary Registers window

The Auxiliary Registers window shows registers for ARCTM processors. The ARC auxiliary register set contains special status and control registers. Use the Auxiliary Registers window to view and change the values of these registers.

Modifying the value of an auxiliary register

To modify the value an auxiliary register:

  1. Click the register value, and then click the Change button. Or Right-click the register value, and then click Change. The Change dialog is displayed.
  2. Modify the value in the New value field of the Change dialog, and then click OK.
