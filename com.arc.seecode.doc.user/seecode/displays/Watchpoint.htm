Watchpoint

Hardware watchpoints stop program execution when access to the watched entity occurs. A watchpoint is much like a breakpoint in that program execution will stop when a watched entity is reached or changes.

Depending on your target, you can set a watchpoint on a machine register or variable using the Registers, Local Variables, or Global Variables window.

Use the Set Watchpoint dialog to set a watchpoint.

Use the Watchpoints window to show and manage watchpoints.
