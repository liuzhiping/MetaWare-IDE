MetaWare Debugger command reference

The following is a list of all debugger command-line commands you can enter using the Command: input field or using the command-line debugger. Variable names are shown in italic, optional command parameters are shown in brackets [|]. For a detailed description of each command and the command language, see the MetaWare Debugger User's Guide.

Command syntax Description
attach pid Attach a running process; pid is the process ID 
break Display current breakpoints 
[l|t]break [filename]!linenumber Set a breakpoint on address determined from filename and linenumber 
[l|t]break address Set a breakpoint on address 
call expr Allows debugging a call of a function
continue [-detach] Start or resume execution (same as run)
delete break index | all  Delete a breakpoint 
delete watch index  Delete a watchpoint
detach  Detach the current process 
dis address Generate a disassembly listing of your program 
disable break index | all Disable a breakpoint  
disable watch index | all Disable a watchpoint 
display display Show display named display -- this is primarily for user-defined displays
doc View online documentation (PDF format)
enable break index | all   Enable a breakpoint 
enable watch index | all  Enable a watchpoint
[modifier] emem address values  Enter values into memory at address  
eval expression  Evaluate or change the value of expression and print the result 
evalq expression  Evaluate or change the value of expression but do not print the result (useful for assignments in scripts) 
exit Quit the debugger 
file Display the name and arguments of the current executable 
file2mem address filename  Write memory from a binary file 
globals [regex]  Display current global variables and functions regex is an optional regular expression 
go address Run program to address  
help all  Display information about all commands 
help summary  Display a summary of all commands 
help topic  Display information about topic, which can be any debugger command or one of the following:
show
step
misc
tip
if (expn) command  If expn evaluates to non-zero, perform the command (no 'else' statement available) 
if ... endif  Nest conditional commands (see the MetaWare Debugger User's Guide for syntax and an explanation of use) 
isi [n] Step into n machine instructions 
iso [n] Step over n machine instructions 
kill  Kill the current running process 
locals  Display current local variables 
log filename [a] [i] [o]  Log commands and their results to filename
[a] appends to filename
[i] logs input commands
[o] logs output results 
log off  Terminate logging and close the log file 
[modifier] mem address Show memory in your program 
mem2file address length filename  Dump memory to a binary file 
memset dest val length   Set memory at dest to val for length bytes 
modules  Display current loaded executable and shared libraries 
print message  Print out message 
print1 message  Print out message to standard out 
read filename  Process commands in filename 
register [bank | regex]  Display machine registers 
restart [args]  Kill the current process and reload
ret [n]  Return from n functions calls, n defaults to 1 
run [-detach] Start or resume execution, same as continue
set [var = value]  Set or display debugger setting. Valid debugger settings are args (to set new program argument values), addr_width (to set the minimum width of hex addresses in certain displays), source_path, dir_xlation, and printlines (to set the number of lines printed by the memory command)
show log   Show a log of all commands and outputs since the debugger was started  
sleep nnn Sleep nnn milliseconds
source [filename]!linenumber  Display filename at linenumber 
source  Display your current source file 
source +N  Display an additional N lines of source
source address  Find source at the address 
source filename  Display filename 
source -N  Display N lines of preceeding source 
ssi [n]  Step into n source statements 
sso [n]  Step over n source statements 
stack  Display the run-time stack 
step [n]  Step into n source statements 
stepi [n]  Step over n source statements 
symbols <executable> [-load=offsets] [-quiet]  Load the symbols from the named executable file
system sys_command  Execute a command on the host system 
trace [signal]  Enable tracing of specified signals 
unset var  Restore a debugger setting to its default
untrace [signal]  Disable tracing of specified signals 
variable N [=init] [, N=[init]]  Declare an integer debugger variable of name N 
watch  Display current watchpoints 
watch address, length [, eval expr][, thread threadno]  Set a watchpoint on address for length bytes 
watch expression [, eval expr][, thread threadno]   Set a watchpoint on an expression 
watchreg reg [, eval expr][, thread threadno]  Set a watchpoint on a register reg
while (expn) command While the expn evaluates to non-zero, perform the command -- expn is anything that can be evaluated with command eval
