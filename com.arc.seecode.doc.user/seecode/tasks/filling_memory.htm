File and Memory Fill Operations

Use the File/Memory/Fill operation dialog to copy a file to memory, copy the contents of memory to a binary file, or fill memory with a specific byte value or string. To access the File/Memory/Fill operation dialog, select Tools | File-Memory-Fill Operation.

You can perform the following tasks using the File/Memory/Fill operation dialog:

  • Copy a file into memory
  • Copy memory to a file
  • Fill memory
See also, About S-record files for information on how to create files for accurately initializing memory using the Motorola S-record format.

Copying a file into memory

You can specify the contents of memory by loading a file you previously saved using Copy memory to a file, or loading an ASCII file in the Motorola S-record format. See About S-record files for more information on how this format can help you more accurately fill memory.

  1. Select Copy a file into memory.
  2. If you are loading an S-record file, select The file is an S-record file.
  3. If you are loading a file previously written out by the debugger, specify the beginning address in the Address field.
  4. Enter or browse to the file containing the values to read using the File to read field.
  5. Click OK to enact the memory copy action.

Copy memory to a file

You can copy the contents of memory to a binary file. You can use this file to initialize memory values at a later time using the copy file to memory operation.
  1. Select Copy memory to file.
  2. Enter the starting address in the Address field.
  3. Enter the total size in bytes of memory you want to copy in the Length field.
  4. Enter or browse to the file where you want to write the memory using the File to write field.
  5. Click OK to save memory to the file.

Filling memory with values

You can fill a block of memory with arbitrary values. You can use this option to initialize larger blocks of memory with the same byte value, hexidecimal string, or character string.
  1. Select Fill memory.
  2. Enter the starting address or memory in the Starting address field.
  3. Enter the total size in bytes of memory you want to copy in the Length field.
  4. Select the type of value or string type that you want to fill memory. The options are:
    • repeated 1-byte value enter a two character value into the value field.
    • repeated 2-byte value enter a four character value into the value field.
    • repeated 4-byte value enter an eight character value into the value field.
    • repeated hex string enter a string to be interpereted as a hexidecimal value into the string field.
    • repeated character string enter a string of characters into the string field.
  5. Click OK to fill memory.
