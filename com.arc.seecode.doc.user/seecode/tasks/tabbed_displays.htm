Adding and removing tabs in windows

Each internal window can hold multiple display windows and shows them as tabs. Click the tabs to switch the current display in the window.

This help topic describes how to use the tabs

  1. Adding a display to a window
  2. Moving a display from one window to another
  3. Removing a display from a window
  4. Making the display into a separate floating window

Adding a tab to a window

To add a tab directly to a window without having to drag a tab:

  1. Right-click in the window
  2. Select Display > displayname from the popup menu

Moving a display from one window to another

To move a display from one window to another: Click and drag the tab or select it's title bar and drag it to the window you want to place it inside.

Removing a tab from a window

To remove a tab from a window:

  • Right-click in the window and select Close from the popup menu.

    OR
  • If you are using the Sash pane style GUI, click the close button on the top left corner of the internal window .

Making an internal window into a floating window (MDI and Sash-pane only)

You can make displays contained in the MDI or Sash-pane style debugger into external floating windows. Floating windows appear as frames in the operating system independent of the main debugger frame. Tabbed displays contained within the floating windows can still be dragged and dropped into other windows.

To make an internal window "floating" right click in the display and select Detach frame.
