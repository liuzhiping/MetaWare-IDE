Using interface features

Several features have been added to the MetaWare debugger to enhance ease of use and allow you to organize and configure your debug session.

Drag and drop tabs in windows

Each internal window is capable of holding multiple displays as tabs within the window. See Adding and removing displays.

  • To move a tab from one window to another, click and drag the tab of the display and drop it on another window.
  • To move a tab out of a tabbed window into its own separate window, click and drag the tab to an empty area in the debug desktop. You can also right click-within the window and select Tear Off Display.
  • To remove a tab from a window without closing the entire window, right click in the window and select Close.

Dockable display regions

You can "dock" a window by dragging it to the edge of the debug frame.

  • To dock a window, click the title bar and drag it to the top, left, right, or bottom of the debug desktop.
  • To remove the window from the dock, click the title bar and drag it away the edge where it is docked.
  • You can enable or disable docking by selecting Tools | Debugger Preferences from the main menu and using the Debugger Preferences dialog.

Snap-to-grid windows

By default, internal windows snap to an invisible grid to make them easy to arrange.

  • To unsnap, select Tools | Debugger Preferences from the main menu and uncheck Windows snap-to-grid.
  • You can enable or disable Snap-to-grid by selecting Tools | Debugger Preferences from the main menu and using the Debugger Preferences dialog.

Window layouts

After you have configured the windows as you like them, you can save the window layout to load again later.

  • To save a window layout to the current project, select Windows | Save Window Layout.
  • To save a window layout separately from the current project, select Window | Save Layout As....
  • To load a saved window layout, select Windows | Open Recent Layout | <layout.windows> or Windows | Open Layout.
  • By default, the debugger saves the window layout to the current project when it closes. You can disable autosave by selecting Tools | Debugger Preferences from the main menu and using the Debugger Preferences dialog.

Debugging Projects

You can create a debug project rather than having to set options each time you run the debugger. See Creating and Editing Projects.
