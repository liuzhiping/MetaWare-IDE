Adding/removing a status bar

Every window has the potential to output status events to a status bar component. For example, the status bar on the Source window will tell you when a breakpoint or watchpoint has been set. The graphic below shows the status bar after a breakpoint has been set on a source statement. The breakpoint number, address, source file, line, function, and compile directory are displayed in the status bar.

To enable the status bar:

  1. Right click in the window you want to add or remove the status bar.
  2. Select Toolbars | Status.

To remove the status bar, right-click and select Toolbars | Status again.
