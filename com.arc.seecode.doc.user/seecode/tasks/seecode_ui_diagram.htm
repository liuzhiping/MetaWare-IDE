Understanding the MetaWare debugger user interface

Stepping toolbar

Used to advance program execution. See Stepping your program.

Program location

Program location indicator show the full path to the program in the current process.

Target system

Target system indicator shows the system that is being debugged upon. Such as simulator, or DLL (hardware). See Setting Target Options.

(Source) Display

The individual internal window with the Source window showing.

Options toolbar

See Options Toolbar.

Command-line

Enter command-line input into the Command: field. Press Enter to commit the command. See Entering Debugger Commands and Command Reference.

Console

The Debug Console provides command output from the debugger. See Debug Console.

Running/Stopped indicator

Indicator shows Run # when the process is executing and Animating when the process is in animation mode..
